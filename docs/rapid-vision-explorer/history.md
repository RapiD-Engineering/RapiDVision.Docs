# History

**History** is Explorer's claim history — a record of every [claim report](reporting.md) you've generated, grouped by month, so you can revisit, re-download, or remove them later.

## Overview

Open **History** from the navigation menu.

- If you haven't generated any claim reports yet, the page shows **"No claim history found"**.
- Otherwise, reports are grouped into cards by month, most recent first, going back up to two years. Each month's card header shows the month and year, and a running **Total** of that month's claim values.

## Reviewing an Entry

Within each month, every generated report appears as its own card, showing:

- **Order** — the Order name the report was generated for.
- **PO Number**
- **Value** — the report's total claim value.
- **Claimed On** — the date the report was generated and marked as claimed.
- **Created By** — who generated it.

## Report and Link Actions

Each card has:

- **Open Claim Report** (PDF icon) — downloads the report again (choose where to save it) and opens it.
- **Copy Claim Link** (clipboard icon) — copies the shareable link to the report's included images to your clipboard. This is shown alongside the link's **Expires on** date, and is only available while the link hasn't expired.
- Once a link has expired, the clipboard button is replaced with **Refresh Link**, which generates a new shareable link — valid for the number of days set in [Settings](settings.md#data) under **Generated Links to images expire after** — and saves it back to the claim.

## Removing an Entry

Right-click a claim card and select **Delete** to remove it from your claim history. You'll be asked to confirm: "Do you want to remove claim {Order name}?"

This only removes the history entry — it doesn't affect the Order or its Order Rules, and it doesn't un-mark them as claimed.
