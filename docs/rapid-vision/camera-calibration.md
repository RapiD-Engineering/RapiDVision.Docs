# Camera Calibration

To measure objects in real-world units (for example, millimeters), you must calibrate the camera.

Calibration is done by placing an object with known dimensions in front of the camera and mapping image pixels to real-world measurements.

## Steps

1. Use a **Classic Vision** configuration.
2. Go to **Settings** and select **Calibrate**.
3. Make sure the canvas is empty before continuing.
4. Place the object with known dimensions in the camera's field of view.
5. Adjust the **Filter Range** slider until only the object remains visible.
6. Select **Camera Calibration**.
7. Enter the known dimensions in the calibration dialog.
8. Select **Calibrate camera** and save the result.
